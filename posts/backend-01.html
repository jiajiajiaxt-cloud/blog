Spring Boot 入门教程

什么是 Spring Boot

Spring Boot 是 Spring 生态系统中的一个项目，它简化了 Spring 应用的开发过程，提供了自动配置和约定优于配置的特性。

主要特点

  • 独立运行的 Spring 应用
  • 内嵌 Servlet 容器
  • 提供自动配置
  • 没有代码生成和 XML 配置

快速开始

创建一个简单的 REST API：

@RestController
public class HelloController {
    @GetMapping("/hello")
    public String hello() {
        return "Hello, World!";
    }
}
